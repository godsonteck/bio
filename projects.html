Online Cross-Platform Car Rental System

A web and mobile platform for booking, managing, and renting cars online. Built for both Android and iOS users, with real-time availability and secure payments. I built this app to make car rental easy, accessible, and secure for everyone.

Key Features:
  • Real-time car availability and booking
  • Secure online payments
  • User account management
  • Admin dashboard for fleet management
  • Push notifications and reminders
Technologies Used: Built with Flutter, Firebase, RESTful APIs, and Web
My Role: Full development, UI/UX design, backend integration
Challenges & Solutions: The main challenge was ensuring seamless sync between web and mobile platforms. I solved this by using Firebase as a real-time backend and designing a unified API layer.
Results: Successfully launched for both Android and iOS, 2,500+ bookings processed
